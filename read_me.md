# Branchpad

Minimalistická offline aplikace pro stromové poznámky.
Bez účtu. Bez cloudu. Bez sledování.

## Co je Branchpad?
Branchpad je jednoduchý nástroj pro psaní a organizaci poznámek ve stromové struktuře.  
Funguje kompletně offline přímo v prohlížeči.
Stačí otevřít soubor `index.html` a začít psát.

## Hlavní vlastnosti
- 🌲 Stromová struktura poznámek
- 🔎 Vyhledávání včetně podpoznámek
- 💾 Automatické ukládání (IndexedDB)
- 📄 Export do Markdown (.md)
- 📦 Export / import JSON
- 📴 Plně offline provoz
- ⌨️ Ovládání klávesnicí

## Jak aplikaci používat
1. Stáhněte nebo naklonujte repozitář.
2. Otevřete soubor `index.html` ve webovém prohlížeči.
3. Začněte psát.

Není potřeba instalace ani připojení k internetu.
Poznámky se ukládají lokálně v prohlížeči (IndexedDB). Pro zálohu použijte export JSON.

## Ukládání dat
Data jsou ukládána lokálně pomocí IndexedDB v prohlížeči.
Pokud vymažete data prohlížeče, dojde ke ztrátě uložených poznámek.  
Doporučuje se pravidelně používat export do JSON jako zálohu.

## Filosofie projektu
Branchpad vznikl jako jednoduchý nástroj pro strukturované myšlení.  
Nesnaží se být komplexním systémem ani cloudovou službou.

Je to nástroj.  
Otevřít. Psát. Hotovo.
